## Асуудал

1 ээс n хүртэлх тоог рекурсив байдлаар олоорой .

## Шийдэл

```
class Soft0404 {
  printNos(n: number) {
    if (n == 0) return;
    this.printNos(n - 1);
    console.log(n);
  }
}
```

## Дүгнэлт

Рекурсив функц дуудаж байхад түүний доторх функц нь үйлдэл хийж дууссан бол тухайн функц үргэлжлүүлнэ .
