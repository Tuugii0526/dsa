Энэхүү функцийн өсөлтийн хурд юу вэ ?

```
class Soft{
    public int solution(int n){
        if(n%2==0)return 0;
        int sum=0;
        for(int i=0;i<=n;i++){
            sum+=i;
        }
        return sum;
    }
}
```

Тогтмол эсвэл n хоёрийн аль нэг нь . Энэ тохиолдолд өсөлтийн хурдыг 3 тохиолдолд салгаж үзнэ .

1. Best case
2. Average case
3. Worst case
   Worst case нь амьдралд ашиглагддаг .
